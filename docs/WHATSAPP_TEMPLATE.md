# التواصل اليدوي عبر WhatsApp

هذه النسخة لا تستخدم قالب Meta ولا WhatsApp Cloud API. الرسالة تُولّد داخل لوحة التحكم ويمكن تعديلها قبل فتح واتساب.

## طريقة العمل

1. راجع رسالة العميل في صفحة التفاصيل.
2. عدّل النص عند الحاجة.
3. اضغط **فتح واتساب**.
4. يفتح `wa.me` أو WhatsApp Web بالحساب المسجل لديك والرسالة مكتوبة مسبقًا.
5. اضغط إرسال بنفسك داخل واتساب.
6. بعد الرجوع للوحة التحكم، اضغط **تم الإرسال** لتسجيل التواصل.

لا يتم تسجيل العميل على أنه `CONTACTED` عند فتح الرابط فقط.

## ضوابط الرسالة

- قصيرة وبالعربية السعودية الطبيعية.
- تتضمن اسم النشاط ورابط المعاينة الخاص.
- لا تدّعي أن العميل طلب الموقع.
- لا تتضمن أسعارًا أو خدمات أو خبرات غير مؤكدة.
- تتضمن اعتذارًا مهذبًا وخيار عدم تكرار التواصل.
- يمنع فتح واتساب إذا كانت الحالة `DO_NOT_CONTACT` أو `NOT_INTERESTED`.
